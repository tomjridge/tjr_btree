Module Tjr_btree__Btree_intf.Root_ops_type

The B-tree code exports a pre_map_ops version of a map, with explicit passing of the reference to the root of the B-tree. In order to implement the map_ops interface, we need to store the "current" reference to the B-tree root in the global state somehow. A value of type ('r,'t) btree_root_ops reveals how to read and write this reference in the global state.

type ('r, 't) btree_root_ops = {
root_ops : ('r, 't) Tjr_monad.with_state;
}
