Module Btree_intf.Blk_allocator_ops_type

A type for managing the free space on the disk.

type ('blk_id, 't) blk_allocator_ops = {
alloc : unit -> ('blk_id, 't) Tjr_monad.m;
free : 'blk_id -> (unit, 't) Tjr_monad.m;
}

NOTE we assume alloc never fails, or that error is handled elsewhere in the monad
